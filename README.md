# 📚 Perpustakaan

Sistem **manajemen perpustakaan** berbasis web untuk mengelola koleksi buku, data anggota, dan transaksi peminjaman.

## ✨ Fitur

- **Manajemen Koleksi Buku** — Tambah, edit, hapus buku
- **Data Anggota** — Pendataan anggota perpustakaan
- **Transaksi Peminjaman** — Catat peminjaman dan pengembalian
- **Pencarian** — Cari buku berdasarkan judul/penulis

## 🛠️ Tech Stack

- **Next.js** — React framework
- **TypeScript** — Type safety
- **Tailwind CSS** — Styling

## 🚀 Cara Menjalankan

```bash
npm install
npm run dev
```
